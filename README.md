# Discout Cards Wallet

## Development

### Requirements

- Node.js and npm

### Setup

1. Clone the repository

```bash
git clone https://github.com/ivteplo/discount-cards-wallet
```

2. Navigate to the folder

```bash
cd discount-cards-wallet
```

3. Install dependencies

```bash
npm install
# if you prefer yarn:
yarn install
```

4. Start the development server

```bash
npm run dev
```
